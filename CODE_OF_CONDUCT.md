# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Scope

The Code of Conduct applies in all project spaces — issues, pull requests, discussions, code review, the wiki, and any community channel that represents the project. It also applies when individuals officially represent the project in public spaces.

## Enforcement

Reports of unacceptable behavior should be sent to the project maintainers through the contact channels listed in [`SECURITY.md`](./SECURITY.md). Reports related to a security vulnerability must instead use [GitHub's private vulnerability reporting](https://github.com/yoda-digital/mcp-gitlab-server/security/advisories/new).

All complaints are reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy and security of the reporter.

## Project-specific notes

This project is built primarily by humans collaborating with AI agents. The expectations of respectful, professional, and good-faith engagement apply equally to:

- Direct human collaboration in issues and PRs.
- Code generated by AI agents and submitted under a contributor's name.
- Triage, review, and discussion activity.

Conduct rules for AI agents working on this codebase live in [`ai_code_of_conduct.md`](./ai_code_of_conduct.md). Those rules supplement, rather than replace, this document.
